## IDEMIA Central Distribution Note

All six states in this gap (AR, DE, IA, KY, NY, WV) use **IDEMIA** as their mDL vendor. IDEMIA does not operate a standalone IACA certificate download portal. Their distribution model:

1. **IDEMIA Verify SDK** — bundles IACA certs (Production/Demo/UAT) for all IDEMIA-backed states. Requires IDEMIA Experience Portal account ([experience.idemia.com](https://experience.idemia.com/)).
2. **AAMVA VICAL** — [vical.dts.aamva.org](https://vical.dts.aamva.org/) — CBOR/COSE signed bundle. Free download with ToS click-through. cipher is re-parsing this now.
3. **Direct state download** — only WV publishes on .gov. Others delegate to IDEMIA/VICAL.

**Recommended acquisition strategy:**
1. Parse VICAL first (cipher's task) — may contain DE, KY, NY certs already
2. Download WV root directly from .gov
3. Send outreach emails for AR, DE, IA, KY, NY only if VICAL doesn't contain their certs
4. Register for IDEMIA Experience Portal as fallback path for all IDEMIA states
